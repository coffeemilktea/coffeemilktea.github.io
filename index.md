# Healthcare Data Tools

A collection of professional tools for working with medical data formats including DICOM and HL7.

## Projects

### 🏥 DICOM Toolbox
**Generate and manage DICOM medical imaging files**

Create, configure, and generate DICOM files with ease. Perfect for testing, development, and learning the DICOM standard.

[**Launch DICOM Toolbox →**](tools/dicom-generator.html)

---

### 📋 HL7 v2.x Parser
**Parse and visualize HL7 clinical messages**

Parse HL7 v2.x messages with detailed segment and field breakdown. Understand your healthcare data with clarity and precision.

[**Launch HL7 Parser →**](tools/msgparser.html)

---

### 🔄 HL7 Diff Checker
**Compare HL7 messages side-by-side**

Identify differences between HL7 messages segment by segment. Perfect for data validation and troubleshooting integration issues.

[**Launch HL7 Diff Checker →**](tools/index.html)

---

## About

These tools are designed for healthcare IT professionals, developers, and students working with medical data standards. Whether you're debugging HL7 messages, testing DICOM implementations, or learning healthcare data formats, you'll find these utilities helpful.

All tools are client-side applications that run entirely in your browser—no data is sent to external servers.
